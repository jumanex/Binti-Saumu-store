# Binti Saumu store

Static e-commerce website for Binti Saumu store. This project includes:

- `index.html` homepage
- `products.html` product listing
- `cart.html` shopping cart
- `checkout.html` checkout flow
- `payment-confirmation.html` payment confirmation page
- `admin-login.html`, `admin.html`, `admin-products.html` for admin product/order management
- `js/script.js` client-side logic
- `css/style.css` styling

## Deploy options

### GitHub Pages
1. Create a new GitHub repository.
2. Add this folder as a local git repo and push.
3. Enable GitHub Pages from repository settings using `main` branch and `/ (root)`.

### Netlify
1. Create a Netlify account.
2. Connect your GitHub repo or drag this folder into Netlify.
3. Set the publish folder to the project root.

### Vercel
1. Create a Vercel account.
2. Import the GitHub repo.
3. Deploy the static site.

## Local preview
Open `index.html` in your browser to preview locally.
